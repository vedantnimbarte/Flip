# Security Policy

## Supported versions

Only the latest release receives security fixes.

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, report privately via [GitHub Security Advisories](https://github.com/vedantnimbarte/dlm/security/advisories/new) or email **nimbartevedant@gmail.com**.

Include what the issue is, how to reproduce it, and the impact. You can expect an acknowledgment within a few days, and a fix or public disclosure will be coordinated with you.
